#To create loadbalancer
=======================
1. create target group choose alteast one instance.

2. Go to load balancer console given name.

3. select atleast 2 availability zones. and must select AZ that have your target group instance.
   (if your target group instance in located in us-east-la then select us-east-1a otherwise loadbancing not point out the instance).

#To create auto scaling
=======================
1. Go to auto scaling console give the name.

2. create lunch templete from existing instance. and select that lunch template.

3. then select the loadbalancing.

4. keep desired 1, min 1, max 5 (important).

5. if instance is working well which is created by ASG then delete the manual instance.
