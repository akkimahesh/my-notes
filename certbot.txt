install first
-------------

sudo apt install certbot python3-certbot-nginx -y

sudo certbot --nginx --register-unsafely-without-email --non-interactive --agree-tos -d example.com


Renewal in Systemd Service
--------------------------
Create /etc/systemd/system/certbot-renew.service:

text:
-----

[Unit]
Description=Certbot Renewal
[Service]
Type=oneshot
ExecStart=/usr/bin/certbot renew --quiet --register-unsafely-without-email
