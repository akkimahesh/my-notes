STEP 1️⃣ Install Certbot
Amazon Linux / RHEL
-----------------
sudo yum install certbot python3-certbot-nginx -y

Ubuntu (if needed)
----------------
sudo apt install certbot python3-certbot-nginx -y

STEP 2️⃣ Ensure HTTP config exists (important)

Your current config should look like this (HTTP only):

📄 /etc/nginx/conf.d/maheshakki.conf

# DEV
server {
    listen 80;
    server_name dev.maheshakki.shop;

    location / {
        proxy_pass http://localhost:3000;
    }
}

# QA
server {
    listen 80;
    server_name qa.maheshakki.shop;

    location / {
        proxy_pass http://localhost:4000;
    }
}

# PROD
server {
    listen 80;
    server_name maheshakki.shop;

    location / {
        proxy_pass http://localhost:8000;
    }
}


Test:

sudo nginx -t
sudo systemctl reload nginx

STEP 3️⃣ Generate SSL certificates (AUTO CONFIG)

Now the easiest part 😄
Certbot will:

Get SSL

Create HTTPS blocks

Add HTTP → HTTPS redirect

Run ONE command:

sudo certbot --nginx \
  -d maheshakki.shop \
  -d dev.maheshakki.shop \
  -d qa.maheshakki.shop


When prompted:

Email → enter your email

Agree TOS → Y

Redirect HTTP → HTTPS → YES (recommended)

STEP 4️⃣ Done ✅ (what Certbot does for you)

Certbot automatically:
