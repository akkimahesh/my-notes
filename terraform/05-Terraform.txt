NAT
---
NETWORK ADDRESS TRANSLATION

nat gateway mainly used for outgoing traffic but not incoming traffic.

(for example server wants to download packages, software so here nat gateway provides internet.)


outgoing traffic: requesting something from the server. (like downloading the packages)
inconing traffic: fetching data from local to server. (like entering into the server)
