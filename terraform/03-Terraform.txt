data-sources
------------
data source block in terraform is mainly responsible for getting right resouce details from infra if dynamic changes happens or not.

syntax for data_source
----------------------
data "<resource_name>" "example" {
          statements
}

for_each
--------
for_each is key word in terraform used for loop items that must be type = map. like "key = value"

To get sequence of items we use "each" key word. for example each.key and each.value

dynamic
--------
dynamic is also used for loop the items but we used it when blocks are repeated inside the resource.

statefile and remote statefile
------------------------------

1.statefile is reposible for storing the resouces details after execute the "terraform apply" command.
